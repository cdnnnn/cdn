Please follow these steps to update:

Step 1: Open PyCharm.
Step 2: Go to Settings (Windows/Linux: File → Settings | macOS: PyCharm → Settings/Preferences).
Step 3: Navigate to Plugins.
Step 4: Locate the Test plugin and click Uninstall.
Step 5: Restart PyCharm if prompted.
Step 6: Open the Plugins marketplace, search for Test, and install the latest version.
Step 7: Restart PyCharm to complete the installation.
